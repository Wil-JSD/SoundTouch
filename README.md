# SoundTouch
SoundTouch deleverables

With the installer (SoundTouchV01-x86_64-1.0.0-Installer) the windows app can be installed. The app run with admin rights.
Android app could be stored (50 Mb) as a release. Also it could be asked for via jsd@kpnmail.nl and wetransfer. 
For questions please mail to jsd@kpnmail.nl.

The cloud directory contains the server directory structure, the server script json_server.py 
and a stations directory containing the <station>.json files and png files (not used yest but could be uses)
This should be stored on a server with Python, e.g. a Raspberry. The application uses
default the server http://raspberry-pi:8081. This could be seen and changed on the settings page to e.g.
http://192.168.1.20:8081 when running the server on your laptop.

How to use the app
1. Web service must be running
2. Configure the cloud service on the settings page if the default setting is incorrect.
3. Select a station on the station page. The station will then become audible.
4. The station can now be added to the presets. This can be done via the Soundtouch itself, via the remote control, or by  returning to the first screen and long-pressing a preset.
5. Six presets can be filled.
6. Add, update or delete stations on the settings screen
7. The Soundtouch can be operated via the app. The app can also be closed, and the Soundtouch can be used via the buttons and/or the remote control.
